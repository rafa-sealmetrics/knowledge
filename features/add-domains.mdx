# How to Add Domains in Your Account

Learn how to add domains to your SealMetrics account for comprehensive, privacy-friendly analytics tracking across multiple websites.

## Understanding Domain Tracking

SealMetrics defines an entrance as a click that meets one of two conditions:

- An **empty referrer** (direct traffic)
- A **referrer from a domain different from your own**

For instance, if a hit has no referrer, SealMetrics interprets it as Direct traffic. If the referrer is from a different domain, like google.com or facebook.com, SealMetrics classifies the hit as a visitor.

### Privacy-Friendly Approach

Although this method isn't as precise as tracking sessions or unique visitors through cookies, it provides a **privacy-friendly approach**. Even if the same user visits the site multiple times from SEO, each instance is counted as a separate visitor.

## Step-by-Step Domain Setup

### Step 1: Access the Monitor Section

Click on the **"Monitor" icon** in the top navigation bar:

![Sealmetrics Sections 2 Pn](/images/sealmetrics-sections-2.png)

You'll see the main navigation with options for:

- Users
- Accounts
- Referrers
- Costs

### Step 2: Go to Accounts and Add an Account

Navigate to the **Accounts** section and click **"\+ New Account"**:

![Sealmetrics Add New Account 2 Pn](/images/sealmetrics-add-new-account-2.png)

The Accounts page shows:

- **Filter options**: Search existing accounts
- **Account list**: Current configured domains
- **Status indicators**: Active/inactive domains
- **Action buttons**: Edit, delete, or manage domains

### Step 3: Add a Name and Domain

In the account creation form, you'll need to provide:

![Sealmetrics Account Name 2 Pn](/images/sealmetrics-account-name-2.png)

#### Required Fields:

- **Account Name**: A descriptive name for your domain (e.g., "Main Website", "Blog", "E-commerce Store")
- **Domain**: The actual domain URL (e.g., example.com, blog.example.com)
- **UTM Parameters**: Optional tracking parameters

## Domain Configuration Best Practices

### Domain Format Examples

- ✅ **Correct**: `example.com`
- ✅ **Correct**: `www.example.com`
- ✅ **Correct**: `subdomain.example.com`
- ❌ **Incorrect**: `https://example.com`
- ❌ **Incorrect**: `example.com/page`

### Multiple Domain Setup

#### For E-commerce Businesses

```
Main Store: shop.example.com
Blog: blog.example.com  
Support: help.example.com
Landing Pages: landing.example.com
```

#### For Multi-brand Companies

```
Brand A: brand-a.com
Brand B: brand-b.com
Corporate: company.com
Events: events.company.com
```

#### For International Sites

```
Global: example.com
US: us.example.com
UK: uk.example.com
DE: de.example.com
```

## Domain Verification Process

### Verification Steps

1. **Add domain** in SealMetrics account
2. **Install tracking code** on the domain
3. **Wait for data collection** (24-48 hours)
4. **Verify traffic appears** in reports
5. **Configure goals** and conversions

### Troubleshooting Common Issues

- **No data showing**: Check tracking code installation
- **Incorrect attribution**: Verify domain configuration
- **Missing referrers**: Review cross-domain setup
- **Duplicate tracking**: Ensure single tracking code per page
