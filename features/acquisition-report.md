With the acquisition sources report, SealMetrics provides an in-depth look at traffic sources. The report displays the source and medium, along with key metrics such as the number of entrances, page views, conversions, micro-conversions, conversion rate, revenue, and cost. If the cost calculator is enabled, users can also calculate the Return on Ad Spend (ROAS) by comparing cost and revenue.



This report offers valuable insights into the performance of each marketing channel. Additionally, filters can be applied to refine the data, allowing for a more detailed analysis. The report can be filtered by medium, campaign, term, device, or operating system.

This approach enables users to identify, as shown in the example, which operating system yields the best results for their marketing campaigns.
