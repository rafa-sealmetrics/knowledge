How does it work?

TheAdministratoruser can add authorized IP addresses from which users can connect to the platform. This means that, in addition to authenticating with ausernameandpassword, users will only be able to access the platform if they are using an IP address approved by the Administrator.



Benefits of this security measure:

Full access control: Ensures only devices connected from authorized IPs can access the platform.Enhanced protection against unauthorized access: Prevents access from untrusted or unknown networks.Robust security level: Complements traditional authentication methods, adding an extra layer of protection to safeguard your data.

Who can use this feature?

This functionality is available forGROWTHaccounts and above.

How to set it up?

The Administrator can manage authorized IPs directly from the platform's settings panel, ensuring that only users meeting this security requirement can log in.
